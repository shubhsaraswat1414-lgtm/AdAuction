# AdAuction - Publisher Network & Ad Platform

## 📖 Project Description
AdAuction is a comprehensive Publisher Network and Ad Platform built with modern web technologies. It provides a robust interface for managing ad inventories, tracking budget history, monitoring live auctions, and analyzing campaign rankings. The platform also includes audience management and delivery route optimization, offering a complete solution for digital advertising operations.

## 🚀 Tech Stack
- **Framework:** React 19
- **Build Tool:** Vite
- **Routing:** React Router v7
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Charts:** Recharts

## ✨ Features
- **Dashboard:** Overview of key metrics and recent activities.
- **Ad Inventory:** Manage and organize available advertising spaces.
- **Budget History:** Track ad spend and financial history.
- **Live Auction:** Real-time monitoring of ad bidding and auctions.
- **Audience Manager:** Segment and manage target audiences.
- **Campaign Ranking:** Analyze and rank ad campaigns based on performance.
- **Publisher Network:** Oversee publisher connections and metrics.
- **Delivery Routes & Storage System:** Advanced routing and storage visualizations.
- **Responsive Design:** Fully responsive layout with a collapsible sidebar and mobile-friendly navigation.

## 🛠️ Setup Instructions

Follow these steps to set up and run the project locally. Another developer can easily get started without additional help.

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd REACT_PROJECT
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

5. **Preview the production build:**
   ```bash
   npm run preview
   ```
